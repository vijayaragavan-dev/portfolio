## ğŸ¨ Customization

### Update Personal Information
Edit the following in `index.html`:
- Name: Find and replace "Vijayaragavan"
- Contact info: Email, phone, social links
- Skills: Adjust percentages in the skills section

### Update Resume
Place your actual resume PDF at: `assets/Vijayaragavan_Resume.pdf`

Edit CSS variables in `css/style.css`:
```css
:root {
    --primary: #00d4ff;
    --java-color: #f8981c;
    --dark-bg: #0a0a0f;
}
```
